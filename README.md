# e-plantShopping: E-commerce Website

This project is a small e-commerce website built using **React** as part of learning **Full-Stack Development** with IBM. The website is now live on **GitHub Pages** and showcases a variety of plants that can be purchased online.

## Features

- **React**
- **Redux**
- **Vite**
- **Responsive design**
- **E-commerce functionality** including Add to Cart and Checkout buttons but excludes Checkout functionality
